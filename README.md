# 🏛 3D迷路メーカー（エジプト遺跡風）

2Dで迷路を描いて、そのまま3Dの一人称視点で探索できるブラウザゲームです。
HTMLファイル1つで動作します（[Three.js](https://threejs.org/) をCDNから読み込むためネット接続が必要）。

## ▶ 今すぐ遊ぶ

**https://altair-77.github.io/maze/**

（ローカルで動かす場合は [`index.html`](index.html) をブラウザで開くだけ）

### 2Dエディタ
- 100×100マスに **壁 / 消去 / スタート / ゴール** をクリック・ドラッグで配置
- **🎲ランダム迷路を生成**（必ずクリア可能な迷路を自動生成）
- **🔍ゴール可能か確認**（BFSで到達判定し、最短ルートを表示）
- 外壁を囲む / 全消去 / ブラシサイズ変更 / 保存・読込

### 3Dゲーム
- **W A S D** で移動、**マウス / ドラッグ / 矢印キー** で視点、**Shift** でダッシュ
- **Esc** でエディタに戻る、**M** でミニマップ表示切替
- 探索した範囲が自動で地図化される（フォグ・オブ・ウォー）
- 壁にはエジプト壁画（ホルスの目・アンク・スカラベ・ファラオ・ラー）
- ごく低確率で **黄金の財宝（レア図像）** が出現
- ゴールには立体の **黄金のファラオ像**。到達するとライトアップして回転！

## 技術メモ
- 壁は `InstancedMesh` でまとめて描画し、100×100でも軽快に動作
- 壁画・財宝テクスチャは Canvas で手続き生成
- ファラオ像は Three.js のプリミティブで組み立て
